# Task-Manager-API-using-Node.js
# 🧾 Task Manager API

A simple backend project built using Node.js 
This project demonstrates how to create a basic REST API server.

---

## 🚀 Features

- Create a server using Node.js
- Handle HTTP requests using Express
- Implement a GET API endpoint
- Return task data in JSON format

---

## 🛠️ Tech Stack

- Node.js
- Express.js

---

## 📌 API Endpoints

### 🔹 GET all tasks
Returns the list of tasks.

GET http://localhost:5000/tasks
🔹 Root route

Checks if server is running.

GET http://localhost:5000/

📤 Response:

Task Manager API is running


How to Run This Project
Install dependencies:
npm install
Start the server:
node server.js
Open browser:
http://localhost:5000
